## Content
There are many things that I have learned that I could do with materialize css after selecting it as my tool, for example, being able to understand the different color usage like gradient and hex that is offered but also including adding simple icons components and turning an image into different shapes. I also explored different areas of css and component it offered such as collections which is a component of materialize that allows one to organize a collection of groups of objects and scale transitions that can make something look smoother when loading. I learned to use materialize on my own by first and most importantly testing out the different features on jsbin and making things out of it using trial and error. For example, one of my codes in jsbin was making header with a collection class:
```html

  <ul class="collection">
    <li class="collection-item avatar">
      <img src="images/yuna.jpg" alt="" class="circle">
      <span class="title">Title</span>
      <p>First Line <br>
         Second Line
      </p>
      <a href="#!" class="secondary-content"><i class="material-icons">grade</i></a>
    </li>
    <li class="collection-item avatar">
      <i class="material-icons circle">folder</i>
      <span class="title">Title</span>
      <p>First Line <br>
         Second Line
      </p>
      <a href="#!" class="secondary-content"><i class="material-icons">grade</i></a>
    </li>
    <li class="collection-item avatar">
      <i class="material-icons circle green">insert_chart</i>
      <span class="title">Title</span>
      <p>First Line <br>
```

Even the basic code setup of materialize:
```html
  <!DOCTYPE html>
  <html>
    <head>
      <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
 <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">
      <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
      <title>Materialize sandbox</title>
    </head>
    <body>

  <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script>
    </body>
  </html>
```
