# Tinker Tools
##### 3/14/26

## Content
The tool that I decided to pick to use for my freedom project was materialize. My top option was Aframe but after tinkering with it, I realized that it wasnt suited for me. With choosing materialize, tinkering with it is very important and so I tinkered with it in Js Bin with many things materialize had to offer. I decided to make something after setting it up:
```html
<!DOCTYPE html>
  <html>
    <head>
      <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
 <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">
      <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
      <title>Materialize sandbox</title>
    </head>

    <body>
      <div class="container-fluid">
      <h1>Hello world</h1>
        <img src=https://www.thepublicdiscourse.com/wp-content/uploads/2021/02/AdobeStock_57390451.jpeg" class= "responsive-img circle">
        <button class="btn waves-effect">More</button>
      </div>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script>
    </body>
  </html>
```
This code resulted in making the image into a circle and adding a more button that changes to have a wave effect when clicked on it. I also changed the color of the background:
```html
<div class="card-panel teal darken-2">Teal darken background.</div>
<a class="waves-effect waves-light btn red darken-1">Dark Red Button</a>
```
along with changing the color of the texts:
```html
<p class="red-text text-darken-4">This text is red darken.</p>
<div class="blue lighten-5">
    <span class="blue-text text-darken-2">This span has blue darken text.</span>
</div>
```
